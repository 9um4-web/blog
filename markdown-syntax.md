# 마크다운 문법 가이드 (포스트 작성용)

이 블로그의 렌더 파이프라인(`lib/domain/render.ts`)이 실제로 지원하는 문법 정리. GFM 기반 + 수식 + 커스텀 디렉티브.

## 기본 GFM (remark-gfm)

- 표(테이블), 취소선 `~~텍스트~~`, 자동 링크(`https://...` 그대로 써도 링크화), 작업 목록 `- [ ]` / `- [x]`
- 각주: `본문[^1]` ... `[^1]: 각주 내용` — 문서 하단에 자동으로 각주 섹션 생성

## 코드블록

일반 마크다운 펜스 \`\`\`언어 로 작성. shiki로 서버에서 하이라이팅됨.

지원 언어: `javascript` `typescript` `jsx` `tsx` `json` `bash` `shell` `powershell` `python` `sql` `yaml` `toml` `html` `css` `markdown` `diff` `docker` `go` `rust` `java` `kotlin` `c` `cpp` `csharp`

목록에 없는 언어나 언어 미지정 블록은 하이라이팅 없이 그대로 표시됨.

### mermaid

\`\`\`mermaid 코드블록은 자동으로 클라이언트에서 lazy 렌더링된 다이어그램으로 바뀜. 렌더 실패 시 원본 코드가 그대로 보임.

## 수식 (remark-math + KaTeX)

- 인라인: `$E = mc^2$`
- 블록: `$$ \int_0^1 x^2 dx $$`

수식 오류 시 빨간색으로 원문이 표시됨.

## 커스텀 디렉티브

`remark-directive` 기반. 컨테이너 디렉티브(`:::`)는 **반드시 닫는 `:::`가 있어야 함** — 안 닫으면 문서 끝까지 그 블록에 먹힘.

### 유튜브 임베드

```
::youtube[dQw4w9WgXcQ]
```
또는
```
::youtube{id=dQw4w9WgXcQ}
```

영상 ID만(URL 전체 아님), 영숫자/`-`/`_` 6~20자.

### 포스트/시리즈 카드 임베드

```
::post{slug=my-post-slug}
::series{id=3}
```

- `slug`는 영소문자/숫자/`-`만
- `id`는 시리즈의 숫자 ID (양의 정수)
- 카드 형태로 렌더됨(제목·요약 등). 존재하지 않는 slug/id면 에러 표시

### 포스트 인라인 링크

```
:postlink[표시할 텍스트]{slug=other-post}
```

문단 안에 자연스럽게 들어가는 텍스트 링크. `/slug`로 링크 생성, 존재 여부는 검증 안 함(잘못되면 그냥 404).

### 콜아웃

```
:::note
내용
:::

:::note[커스텀 제목]
내용
:::
```

종류: `note`(노트) · `info`(정보) · `tip`(팁) · `warning`(주의) · `danger`(위험)

대괄호로 제목 커스터마이즈 가능, 생략 시 종류별 기본 제목 사용.

### 정렬

```
:::center
가운데 정렬될 내용
:::

:::right
오른쪽 정렬될 내용
:::
```

### 들여쓰기

```
:::indent{n=2}
들여써질 내용
:::
```

`n`은 1~8 (단계당 1.5rem). 생략 시 1.

### 섹션 소속 오버라이드 (fold)

포스트는 h2~h6 제목 기준으로 접기 가능한 `<section>`으로 자동 래핑됨. 특정 블록을 원래 위치의 접기 대상에서 빼내고 싶을 때 사용.

```
:::fold{h=2}
이 블록은 가장 가까운 h2 섹션 직속으로 끌어올려짐
:::

:::fold{h=none}
이 블록은 어떤 접기에도 영향받지 않음 (최상위 고정)
:::
```

`h`는 2~6 또는 `none`. 유효하지 않은 값은 `none`으로 취급. 지정한 레벨 이하의 가장 가까운 조상 섹션으로 끌어올려지며, 이후 등장하는 제목에는 구조적으로 다시 소속될 수 없음(끌어올려진 시점 이후 위치에 고정).

## 주의사항

- 컨테이너 디렉티브(`:::note`, `:::center`, `:::indent`, `:::fold`)는 닫는 `:::` 필수
- `::youtube`, `::post`, `::series`는 leaf 디렉티브라 닫을 필요 없음(한 줄)
- `:postlink`는 텍스트 디렉티브라 문단 안에 인라인으로 사용
- 링크/이미지 URL은 `http`, `https`, `mailto`, `tel` 스킴만 허용 (그 외는 렌더 시 제거됨)
